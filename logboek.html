Week Type Naam Korte omschrijving
2 Evaluatie LockDownBrowser Het installeren van LockDownBrowser zodat we deze kunnen gebruiken voor pe's en examens.
Hier heb ik niets bij kunnen leren omdat er alleen een programma moest geïnstalleerd worden.
3 Opdracht Scratch Een simpel spel maken met scratch waar een ballon naar boven vliegt en vogels moet vermijden.
Hier heb ik basis programmeer logica bijgeleerd, De uitdaging hier was het programma scratch zelf leren te kennen.
4 Evaluatie Git & GitHub Een eenvoudige HTML opdracht die met Git moet geüpload worden.
Mijn grootste uitdaging hier was dat ik de opdracht niet goed heb begrepen/gelezen. Hier heb ik dus bijgeleerd om meer tijd te nemen om de opgave goed te lezen.
4 Opdracht Reflecteren Een reflectie over hoe de Scratch-opdracht ging.
5 Evaluatie Gastsprekers Een korte reflectie over de gastsprekers die zijn langsgeweest.
5 Opdracht Planning Een gedetailleerde semester- en weekplanning voor het semester.
Ik heb geleerd hoe ik een goede week en semester planning kan maken. Ik had het wat moeilijk met het nadenken over wat ik wanneer best kan inplannen.
7 Evaluatie C# Console Een consoleapplicatie in C# met variabelen, controle- en herhalingsstructuren en methodes.
Ik heb bijgeleerd hoe je methodes beter kan aanmaken.
7 Opdracht Carrièrekompas Een vacatureopdracht over werkveldverkenning.
8 Opdracht Feedback C# Console De algemene consoleapplicatie feedback vertalen naar eigen project.
9 Evaluatie HTML & CSS Een webpagina zo goed mogelijk nabouwen met HTML en CSS.
Mijn uitdagingen waren positionering en tijd. Ik heb geleerd om meer tijd te gebruiken voor de belangrijkste elementen.
10 Evaluatie Debugging De fouten uit een bestaande consoleapplicatie halen met de debugging tools van Visual Studio.
Ik heb de debugging tools leren gebruiken voor meerdere fouten op te lossen.
10 Opdracht Feedback debugging De algemene debugging feedback vertalen naar eigen PE.
11 Evaluatie SDG's Een korte reflectie over SDG's.
12 Opdracht Curriculum Vitae Een volledig CV gemaakt met HTML en CSS.
Ik heb CSS beter leren kennen met deze opdracht. Ik had nog altijd wat problemen met de positionering
13 Evaluatie C# WPF De GUI van een WPF-applicatie zo goed mogelijk na bouwen.
Ik heb beter geleerd hoe om te gaan met de WPF tools
13 Opdracht Portfolio Een gedetailleerd portfolio maken met HTML en CSS.
Ik heb meer geleerd over HTML en CSS, positioneren was nog een werkpunt maar gaat al veel beter. De grootste uitdaging was de zelf reflectie.
14 Evaluatie Inhaalmoment Inhaalmoment voor studenten die gewettigd afwezig waren tijdens één of meerdere evaluaties
